# VitaSync ZMR Engine (Frontend)

## Overview
VitaSync ZMR Engine is an AI-based healthcare system designed to help users find reliable hospitals based on ICU, oxygen, and emergency services.

## My Contribution
I designed and developed the complete frontend UI of the project.

## Features
- Hospital search system
- AI assistant simulation
- Trust score visualization
- Reports dashboard
- Interactive hospital map

## Technologies Used
- HTML
- CSS
- JavaScript
- Leaflet.js

## How to Run
Open `index.html` in your browser

## Note
This repository contains frontend only. Backend was developed separately.
